## Setup

### 1. Clone Repository
```bash
git clone https://github.com/nandix-org/nandix-flutter-app.git
cd nandix-flutter-app
```

### 2. Configure Firebase
```bash
flutterfire configure --project=nandix-dc0d8
```

### 3. Install Dependencies
```bash
flutter pub get
```

### 4. Run
```bash
flutter run
```
